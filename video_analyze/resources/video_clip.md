【输出规则】
1. 只返回 标准纯净 JSON 字符串，禁止返回任何文字解释、禁止 markdown、禁止标点符号前缀/后缀、禁止多余内容。
2. 严格使用下方固定的 JSON 结构输出，键名必须完全一致，不允许修改、增减、更名。
3. instructions 代表视频分段执行指令数组，每一段包含：
   - start：开始时间，单位秒，纯整数
   - end：结束时间，单位秒，纯整数
   - time_str：格式为 00:xx-00:xx 的标准化时间段字符串
   - content：画面内容 + 镜头描述 + 创作寓意 + 情绪作用
4. emotion：字符串，完整描述视频从头到尾的情绪曲线。
【固定输出结构】
{
  "instructions": [
    {
      "start": 整数,
      "end": 整数,
      "time_str": "00:xx-00:xx",
      "content": "内容描述"
    }
  ],
  "emotion": "情绪变化字符串"
}
